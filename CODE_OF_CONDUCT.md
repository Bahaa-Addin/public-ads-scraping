# Code of Conduct

## Our Standard

We want this project to be a respectful, practical, and welcoming place to collaborate. Contributors are expected to:

- Be kind, clear, and constructive in discussions and reviews.
- Assume good intent while still giving honest technical feedback.
- Focus on the work, not the person behind it.
- Share context generously so other contributors can succeed.

## Unacceptable Behavior

The following are not acceptable in project spaces:

- Harassment, discrimination, or demeaning language.
- Personal attacks, intimidation, or threats.
- Publishing someone else's private information without permission.
- Deliberately disruptive behavior that blocks collaborative work.

## Enforcement

Project maintainers may edit, remove, or reject content that violates this code of conduct, and may limit participation when needed to protect the community.

If you experience or witness a serious issue, contact the maintainers through a private channel provided by the hosting platform rather than opening a public thread.
